# Menu Interativo PUC-PR

## Descrição
Este é um projeto de menu interativo desenvolvido para a Pontifícia Universidade Católica do Paraná (PUC-PR). O sistema foi desenvolvido de forma modular, dividido em quatro arquivos principais, cada um responsável por uma funcionalidade específica.

## Estrutura do Projeto
O projeto está organizado nos seguintes arquivos:
- `MenuPucPr.py` - Arquivo principal para execução do programa
- `DataManagement.py` - Arquivo que gerencia os dados cadastrados
- `Interações.py` - Arquivo que gerencia mensagens de erro ou sucesso, além de estilizar com a função cabeçalho
- `Operações.py` - Arquivo que contém as operações utilizadas no Menu (Incluir, Listar, Atualizar, Excluir)

## Como Executar
1. Certifique-se de ter o Python instalado em seu computador
2. Navegue até o diretório do projeto
3. Execute o arquivo principal:
   ```
   python MenuPucPr.py
   ```

## Requisitos
- Python 3.13 (Mínimo)
